# Mocky 測試文件夾

本資料夾包含 Mocky 專案的測試資源。

## 📁 目錄結構

```
tests/
├── README.md                    # 本說明文件
├── MANUAL_TEST_SCRIPT.md        # 完整手動測試腳本 (87 案例)
├── SMOKE_TEST.md                # 快速驗收測試清單
└── e2e/
    └── app.spec.ts              # Playwright E2E 測試 (框架)
```

## 📋 測試類型

### 1. 手動測試 (Manual Testing)

- **檔案**: `MANUAL_TEST_SCRIPT.md`
- **用途**: 完整功能驗證
- **案例數**: 87 個測試案例
- **預計時間**: 2-3 小時

### 2. 快速驗收 (Smoke Test)

- **檔案**: `SMOKE_TEST.md`
- **用途**: 發版前快速驗證
- **預計時間**: 10-15 分鐘

### 3. E2E 自動化測試 (未來擴充)

- **檔案**: `e2e/app.spec.ts`
- **用途**: 自動化回歸測試
- **框架**: Playwright

## 🚀 快速開始

### 執行手動測試

1. 開啟 `MANUAL_TEST_SCRIPT.md`
2. 按照每個測試案例逐步執行
3. 記錄結果（✅ 通過 / ❌ 失敗）

### 執行快速驗收

1. 開啟 `SMOKE_TEST.md`
2. 依序完成核心功能檢查清單
3. 確認所有項目通過後發版

### E2E 自動化測試 (需安裝依賴)

```bash
# 安裝 Playwright
pnpm add -D playwright @playwright/test

# 執行測試
npx playwright test
```
