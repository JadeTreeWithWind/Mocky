# Mocky 快速驗收測試清單

**用途**: 每次發版前的快速驗收 (Smoke Test)
**預計時間**: 10-15 分鐘

---

## ✅ 快速驗收清單

### 1. 應用程式啟動 (2 分鐘)

- [ ] 應用程式可正常啟動
- [ ] 主視窗佈局正確 (標題列、側邊欄、主內容區、狀態列)
- [ ] 歡迎頁面正確顯示

### 2. 專案建立 (2 分鐘)

- [ ] 點擊 "+" 可開啟建立專案 Modal
- [ ] 填寫名稱後可成功建立專案
- [ ] 新專案出現在側邊欄
- [ ] 自動導航至專案詳情頁

### 3. 路由管理 (3 分鐘)

- [ ] 點擊 "+" 可新增路由
- [ ] 路由列表正確顯示
- [ ] 點擊路由可開啟編輯器
- [ ] 可刪除路由

### 4. 路由編輯器 (3 分鐘)

- [ ] Method 選擇器可切換 HTTP 方法
- [ ] Path 輸入正常
- [ ] Status Code 可修改
- [ ] JSON 編輯器可輸入內容
- [ ] 儲存狀態顯示正確

### 5. Mock Server (3 分鐘)

- [ ] 點擊 "Start Server" 可啟動伺服器
- [ ] 使用瀏覽器訪問 API 回傳正確 JSON
- [ ] 點擊 "Stop Server" 可停止伺服器

### 6. 資料持久化 (2 分鐘)

- [ ] 關閉並重啟應用程式
- [ ] 專案列表保留
- [ ] 路由內容保留

---

## 🚀 快速測試指令

```bash
# 開發模式測試
pnpm run dev

# 類型檢查
pnpm run typecheck

# Lint 檢查
pnpm run lint

# 建置測試
pnpm run build
```

---

## 📋 API 快速測試

測試伺服器啟動後，使用以下 curl 指令驗證：

```bash
# GET 測試
curl http://localhost:8000/users

# POST 測試
curl -X POST http://localhost:8000/users -H "Content-Type: application/json" -d "{}"

# 檢查狀態碼
curl -I http://localhost:8000/users
```
